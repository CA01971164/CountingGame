# 数当てゲーム

## サービスの概要

コンピュータが生成した数を当てるゲームである。

## このアプリを作ったきっかけ

自分たちの学科で習っている技術で、ゲームを含めたウェブアプリを作ることができることを知ってくて作りました。加えて、自分がこのゲームをして楽しかったのは言うまでもない。
